# Smartphone-Sentiment-Pain-Point-Intelligence
NLP-powered sentiment analysis pipeline that scrapes 12,000+ Reddit posts and 8,000+ Amazon/Flipkart reviews using PRAW &amp; BeautifulSoup, classifies user pain points across 6 categories via VADER + TF-IDF/LDA topic modeling, and visualizes brand-vs-brand sentiment trends on an interactive Power BI dashboard.
